# Documents Scaffold

This scaffold generates useful documents for your repos, like `LICENSE.md`, `CONTRIBUTING.md` etc.

## `construct`

Asks for every item it can generate if you want to generate it.

## `patch:add_license`

Adds a `LICENSE.md` to your project.

**Values**:

- `license`: Which license to use.
- `author`: Name to add to the license.
